Description

This module defines the finally tagless Solr DSL. This style admits multiple interpreters, two of which (lazy Texts and an initial encoding) are provided by this library, in the Solr.Query and Solr.Query.Initial modules, respectively.

Ordinary users should not normally have to import this module.
