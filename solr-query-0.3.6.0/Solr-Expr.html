Expr :: SolrType -> *

An opaque Solr expression, indexed by its SolrType. Its interpretation, accessed via e.g. compile, is a lazy Text.

For an initially-encoded version, see Solr.Expr.Initial.Untyped or Solr.Expr.Initial.Typed.
