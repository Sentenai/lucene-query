Description

An initial encoding of a Solr query. This is an alternative interpretation of the Solr language that is more amenable to parsing from arbitrary user input and applying query transformations.

Reinterpretation

reinterpretSolrQuery :: forall expr query. (SolrQuerySYM expr query, HasParamDefaultField query, HasParamOp query, Semigroup (query expr)) => SolrQuery SolrExpr -> query expr Source

Reinterpret an initially-encoded SolrQuery to some other interpretation that supports all of 'SolrQuery'\'s params.

This may be useful for reinterpreting a SolrQuery as a lazy ByteString after it's been type checked and factored with the machinery in this module.
