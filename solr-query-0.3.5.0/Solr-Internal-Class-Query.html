Description

This module defines the finally tagless Solr DSL. This style admits multiple interpreters, two of which (lazy ByteStrings and an initial encoding) are provided by this library, in the Solr.Query and Solr.Query.Initial modules, respectively.

Ordinary users should not normally have to import this module.

Associated Types

data ParamKey query :: * -> *

Different queries support different sets of local parameters. Each parameter is indexed by the type of its value.

Example:

data ParamKey MyQuery a where
  Foo :: ParamKey MyQuery Int
  Bar :: ParamKey MyQuery Bool
  Baz :: ParamKey MyQuery String

(^=:) :: query expr -> Float -> query expr infixr 4

The '^=' constant score operator.

This is given right-fixity to reject queries like q ^= 1 ^= 2.

>>> "foo" =: word "bar" ^=: 3.5 :: SolrQuery SolrExpr
q=foo:bar^=3.5
